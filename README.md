# FireFit Workout Tracker

**Description:**  
FireFit is a powerful and easy-to-use workout tracking app designed to help users stay consistent, monitor progress, and reach their fitness goals. Whether you're training for performance or just staying active, FireFit allows you to log workouts, track improvements, and stay motivated every step of the way.

In today society it can be a little difficult for many individuals to consistently track their workouts, times and progress in one place. As a student-athlete I can definetly relate to this issue, as many athletes either forget their stats or are unable to effectively analyze and track the improvements overtime. This project will there by solve that problem as by creating a simple application that allows users to log workouts, visualize progress and track running and weight journey

As we start to build our app, I plan to create a fitness performance tracker using python. This app will allow user to input their daily workout (time, type of workout, and distance). The data therefor will be stored and displayed through graphs, weekly to ensure that users are give accurate data, which will be able to evaluate their performance and provide positive feedbadcks and possibly suggestions to their users. Some tutorials that I will possibly be using are Tutorials I may use are Python basics: https://www.w3schools.com/python/ and Matplotlib tutorial: https://matplotlib.org/stable/tutorials/index.html . 

Some features that this App will be  providing for their users that can be accessible will are:
Add and save workout data
View personal records (PRs)
Graph progress 
Simple menu-based interface
editting or deleting entries
Weight watcher
